# ChatGPT GUI

A simple demo application ChatGPT using Tkinter package.

### Requirements
```bash
sudo apt-get install python3-tk

python3 -m venv chat_venv
source chat_venv/bin/activate
pip install -r requirement.txt
python gui.py
```
